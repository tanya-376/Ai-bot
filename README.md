# Ai-bot
Mon bot que travaille avec AI
Этот телеграмм бот работает с помощью питона и модели обученной в гугл-машине.
Сначала я обучила модель отличать голубя от синицы, затем экспортировала эту модель и добавила её в вс код, там я подключила нужную функцию к основному коду для бота.
Теперь когда пользователь отправляет в чат с ботом какую-либо фотографию, бот определяет голубь на ней или синица, или не то и не другое.И выводит пользователю результат в виде сообщения.Также указывая на сколько процентов он в этом уверен.
